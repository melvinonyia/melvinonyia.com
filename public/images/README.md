# Error page illustrations

Two files are referenced by the 404 + 500 views but are not committed
because the artwork is not in this repo. Drop them in here and they'll
render automatically.

- `error-404.svg` — illustration for the "Sorry, the page is missing" view
- `error-500.svg` — illustration for the "Sorry, you found a glitch" view

If a file is missing, the `<img>` falls back to an empty image (the
`alt=""` keeps it silent for screen readers).
